Gatewaystack

Open-source trust & control plane toolkit for LLM and agentic apps

Gatewaystack is a modular toolkit for building the trust, identity, and governance layer of your AI stack — the control plane that sits between LLM runtimes and tools (ChatGPT Apps SDK, MCP, etc.) and your backend (Cloud Run, Firestore, APIs).

View Gatewaystack on GitHub
Overview · Packages · Architecture · Security & Governance

As LLM apps evolve into agentic systems with tools, memory, and workflows, the missing piece is a shared layer for identity, policy, and governance. Every project re-implements it: JWT validation, scopes, routing, audit hooks.

Gatewaystack turns that repeated work into a set of composable building blocks: an opinionated Trust Layer (auth, identity, scopes) and a growing Governance Layer (observability, routing, cost, limits) that you can wire into LLM runtimes like ChatGPT Apps SDK and Anthropic MCP.

Use Gatewaystack as the foundation for your own LLM Control Plane or Agentic Control Plane, instead of rebuilding the plumbing for every app and connector.

Gatewaystack building blocks

  • OIDC / OAuth2 integration
  • RS256 JWT verification
  • User identity & session binding
  • Per-tool scopes & consent
  • User-scoped data access
  • Apps SDK / MCP connectors
  • Routing & policy hooks
  • Cloud Run / Firestore friendly
